## Javascript calculations and other functions

[calcAge](https://github.com/mehtajr-prolocity/FormAssembly-Snippets/blob/main/javascript/calcAge.md): allows the user to input a date using a standard date field, and then returns the age in whole years (i.e. no decimals).
